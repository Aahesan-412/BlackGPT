# 🚀 Job Preparation with AI — Intelligent Adaptive Chatbot

A state-of-the-art, fully responsive AI-powered chatbot dashboard engineered to streamline career development and job search preparation. This application bridges the gap between candidates and their dream roles by offering real-time technical practice, HR mock interviews, portfolio-grade resume analysis, interactive aptitude assessments, and tailored career advisory services.

Built with a highly optimized **Python (Flask)** backend and a pixel-perfect, modern dark-themed **Vanilla JS & CSS3** frontend, the platform replicates the fluid streaming interface and features of leading consumer LLM applications (like ChatGPT & Claude).

---

## 🌟 Key Features

### 1. 💼 AI-Powered Career Modules
*   **Resume Analysis (ATS Optimization):** Upload/input resume data to get instant keyword suggestions, formatting critiques, and tailored optimization recommendations.
*   **Dynamic HR Interviews:** Experience realistic mock behavior interviews based on target roles, with situational feedback and body-language simulation notes.
*   **Technical Practice Arena:** Deep-dive into technical mock questions across major frameworks and programming stacks, featuring code evaluation.
*   **Adaptive Aptitude & MCQ Tests:** Test logic, quantitative reasoning, and data interpretation skills with real-time feedback and step-by-step explanations.
*   **Career Path Advisor:** Input your interests and skills to get roadmap generation, industry trend reports, and skill-gap analyses.

### 2. ⚡ Advanced Conversational Engine
*   **Dynamic Token Streaming:** Built-in asynchronous streaming reader on the frontend, rendering tokens instantly as they are processed by the LLM.
*   **Smart Conversation Titles:** The backend automatically analyzes the user’s first prompt and assistant response to construct a concise, context-aware 3-to-5-word chat title.
*   **Full Context Retention:** Integrated server-side state tracking ensures logical continuity over multi-turn conversations.

### 3. 📱 Modern, Responsive Interface
*   **Fluid Responsive Design:** Completely optimized viewport mechanics (`dvh` CSS configurations) to ensure the interface looks stunning and floats seamlessly across all Desktop, Tablet, and Mobile screens.
*   **Sidebar & Chat History Panel:** Switch, rename, delete, and persist chat conversations using integrated local storage sync.
*   **Rich Input Controls:** Features dynamic input expansion, file attachment previews, and **Speech-to-Text (Web Speech API)** voice input.

---

## 🛠️ Tech Stack & Architecture

### **Frontend**
*   **Core:** Semantic HTML5, Vanilla JavaScript (ES6+), Advanced CSS3 Grid & Flexbox
*   **Icons & Fonts:** Space Grotesk (headings), Inter (body), Unicode iconography
*   **Voice Recognition:** HTML5 Web Speech API
*   **Storage:** LocalStorage (Session management & persistence)

### **Backend**
*   **Framework:** Python 3.10+ / Flask
*   **Orchestration:** LangChain / LangChain-Community
*   **AI Integration:** LLMs via Groq Cloud API / OpenAI API / Gemini API
*   **Data Formats:** Stream-compatible JSON chunks
